## 已知限制(設計取捨,非缺陷)

- LINE 重送 webhook 時可能寫入重複資料(MVP 不去重)
- 匿名代號(用戶A/B)每日重排,跨日不固定(尚無使用者名稱表)
- 停機超過一天只補「昨天」的日報,更早的不補
- bot 依賴本機電腦開機 + ngrok 通道存活
- appsettings.json 內含真實 Channel Secret / Access Token,**上 GitHub 前必須換回 placeholder 並處理 .gitignore**
